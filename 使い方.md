# Bruker TOF Reader - セットアップと使い方

MSスペクトルのピーク選択をブラウザで行うツールです。

---

## 1. Pythonのインストール

**Mac:**
```
brew install python3
```
Homebrewがない場合は https://brew.sh からインストールしてください。

**Windows:**
Microsoft Store で「Python」と検索してインストールしてください。

---

## 2. 必要なライブラリのインストール

ターミナル（Mac）またはコマンドプロンプト（Windows）で実行してください。

```
pip install numpy
```

---

## 3. ツールのダウンロード

```
git clone https://github.com/nasu802/bruker_tof_reader.git
```

git がない場合は GitHub ページの「Code → Download ZIP」からダウンロードして解凍してください。

---

## 4. 使い方

```
python3 peak_picker_html.py データフォルダのパス
```

例：
```
python3 peak_picker_html.py data/sample1
```

実行するとブラウザが開き、スペクトルが表示されます。

---

## 5. アップデート方法

git clone でダウンロードした場合、以下の1コマンドで最新版に更新できます。

```
git pull
```

---

## Windows版について

`windows/` フォルダ内の `peak_picker_html.py` を使用してください。
