Bruker TOF Peak Picker — セットアップ＆実行マニュアル

コマンドは「コピー」ボタンでクリップボードに入ります。上から順に実行してください。

▶ デモを見る
→ GitHub: nasu802/bruker_tof_reader
1

Pythonのインストール

Homebrewが入っている場合はターミナルで実行。

brew install python
Homebrewがない場合は brew.sh からインストールしてください。

Microsoft Store で「Python」と検索してインストール。

インストール後、コマンドプロンプトで python --version が表示されればOK。
2

必要ライブラリのインストール（NumPy）

python3 -m pip install numpy
python -m pip install numpy
うまくいかない場合は先に python -m pip install --upgrade pip を実行してみてください。
3

ツールのダウンロード

git clone https://github.com/nasu802/bruker_tof_reader.git cd bruker_tof_reader

以下のページを開き、Code → Download ZIP を選んでZIPを解凍。

https://github.com/nasu802/bruker_tof_reader
  • 解凍後の bruker_tof_reader フォルダを開く
  • 以降のコマンドはそのフォルダ内で実行する
4

実行

基本形
python3 peak_picker_html.py データフォルダのパス
例
python3 peak_picker_html.py data/sample1
基本形
python peak_picker_html.py データフォルダのパス
例
python peak_picker_html.py data\sample1
実行するとブラウザが自動で開き、スペクトルが表示されます。
💡 毎回長いコマンドを打つのが面倒な場合は、mal などの短いエイリアスを設定できます。設定方法はAIに「ターミナルでエイリアスを設定する方法」と聞いてみてください。
5

アップデート（git clone した場合）

以下の1コマンドで最新版に更新できます。

git pull
ZIPで配布した場合は最新ZIPを再ダウンロードしてください。
?

うまくいかないとき

  • python / python3 が見つからない：インストール後にターミナルを再起動してみてください。
  • pip が見つからない：python -m pip install --upgrade pip を先に実行。
  • データフォルダが見つからないと言われる：パスの階層が正しいか確認。スペースがある場合はパスを "..." で囲む。
